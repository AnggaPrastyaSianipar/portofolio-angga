## 📜 Deskripsi Pelatihan Kompetensi

Sertifikat dengan nomor **193115171040-2/VSGA/BLSDM.Komdigi/2025** ini diterbitkan oleh **Kementerian Komunikasi dan Digital (KOMDIGI)** melalui **BBPSDMP Medan**. Kredensial ini merupakan pengakuan formal atas penyelesaian pelatihan intensif selama **24 Jam Pelatihan** pada program **Vocational School Graduate Academy (VSGA) Digital Talent Scholarship 2025** untuk kualifikasi sebagai **Associate Data Scientist**.

---

### 🏛️ Kompetensi Inti yang Divalidasi
Berdasarkan materi yang telah diselesaikan selama total 24 Jam Pelatihan:
* **Konsep Dasar & Tools**: Menguasai konsep dasar Data Science serta penggunaan tools Data Science.
* **Manajemen Data**: Melakukan pengumpulan data, penelaahan data, pemvalidasian data, penentuan objek data, pembersihan data, pengkonstruksian data, hingga penentuan label data.
* **Pemodelan & Evaluasi**: Membangun model prediktif (Machine Learning) serta mengevaluasi hasil pemodelan dengan tepat.

> [!SUCCESS] Otentisitas Kredensial
> Sertifikat ini ditetapkan di Jakarta pada **20 Desember 2025** dan ditandatangani secara elektronik oleh **Bonifasius W. Pudjianto** selaku Kepala Badan Pengembangan SDM KOMDIGI, menggunakan sertifikat elektronik resmi yang diterbitkan oleh Balai Sertifikasi Elektronik (BSrE), Badan Siber dan Sandi Negara (BSSN).

    function updateSlider(index) {
      currentIdx = index;
      if(sliderWrapper) sliderWrapper.style.transform = `translateX(-${currentIdx * 100}%)`;

        if (newIdx >= imagesList.length) newIdx = 0;
        updateSlider(newIdx);
      });
    }

    // Pengaturan Lightbox
    function updateLightboxImage(index) {
      currentIdx = index;
      lightboxImg.src = imagesList[currentIdx];
    }

    function openLightbox(index) {
      updateLightboxImage(index);
      lightbox.style.display = 'flex';
      setTimeout(() => { lightbox.style.opacity = '1'; }, 10);
      document.body.style.overflow = 'hidden';
    }

    const closeLightbox = function() {
      lightbox.style.opacity = '0';
      setTimeout(() => { 
        lightbox.style.display = 'none';

      }, 200);
    };

      img.addEventListener('click', function() { openLightbox(index); });
    });

    });

      e.stopPropagation();
      let newIdx = currentIdx + 1;
      if (newIdx >= imagesList.length) newIdx = 0;
      updateLightboxImage(newIdx);
    });

    closeBtn.addEventListener('click', closeLightbox);
    lightbox.addEventListener('click', function(e) {
      if (e.target === lightbox || e.target === lightboxImg) { closeLightbox(); }
    });

    document.addEventListener('keydown', function(e) {
      if (lightbox.style.display === 'flex') {
        if (e.key === "Escape") closeLightbox();
        if (e.key === "ArrowLeft") prevBtn.click();
        if (e.key === "ArrowRight") nextBtn.click();
      } else {
        if (e.key === "ArrowLeft" && sliderPrev) sliderPrev.click();
        if (e.key === "ArrowRight" && sliderNext) sliderNext.click();
      }
    });
  });
