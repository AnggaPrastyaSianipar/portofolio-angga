## Deskripsi Sertifikasi Kompetensi

Sertifikat dengan nomor **OLZ06L0ORZ65** ini diterbitkan oleh **Dicoding Indonesia** sebagai bagian dari program pelatihan divalidasi Prakerja. Kredensial ini merupakan pengakuan formal yang mengacu pada **Standar Okupasi Pengembang Software (Kode Okupasi KBJI: 2512.03, Indotask: 2512)**, menyatakan bahwa saya kompeten dalam merencanakan, menganalisis, serta memodifikasi aplikasi perangkat lunak sesuai dengan spesifikasi fungsionalitas industri.

---

### Kompetensi Inti yang Divalidasi
Pelatihan intensif sepanjang **9 Jam** ini memvalidasi kapabilitas teknis dan sikap kerja di bidang:
* **Analisis Persyaratan & Alur Kerja**: Mampu meneliti dan mengevaluasi kebutuhan aplikasi dari sisi pengguna, serta merancang perencanaan modifikasi melalui dokumen *requirement* dan diagram alur (*flowchart*).
* **Konsep Dasar Pemrograman & JavaScript**: Menguasai teori sintaksis bahasa pemrograman, variabel, tipe data, logika komputer, serta implementasi penulisan *pseudocode* dan kode program berbasis JavaScript (ES6).
* **Modifikasi Antarmuka Perangkat Lunak**: Mahir mengimplementasikan bahasa markah HTML5 dan pemformatan CSS3 untuk membangun serta mengubah tampilan antarmuka aplikasi web agar lebih responsif dan fungsional.
* **Dokumentasi Teknis & Pengarsipan**: Memahami alur pengarsipan perangkat lunak, penerapan gaya penulisan kode standar (*code style guide*), penulisan komentar yang efektif, serta penyusunan dokumentasi teknis untuk kebutuhan koordinasi dengan *stakeholders*.

        </div>
      `;
      
      // Inject struktur slider baru menggantikan kontainer gambar lama
      themeContainer.innerHTML = sliderHTML;
    }

    // Mengambil referensi elemen DOM slider baru hasil injeksi
    const sliderWrapper = document.querySelector('.cert-slider-slides');
    const slideImages = document.querySelectorAll('.cert-slide-img');
    const sliderPrev = document.getElementById('slider-prev');
    const sliderNext = document.getElementById('slider-next');
    const sliderBadge = document.getElementById('slider-badge');

    // Referensi komponen DOM Lightbox (Mode Teater)
    const lightbox = document.getElementById('custom-light-box');
    const lightboxImg = document.getElementById('light-box-img');
    const closeBtn = document.getElementById('close-light-box');
    const prevBtn = document.getElementById('prev-light-box');
    const nextBtn = document.getElementById('next-light-box');

    // Logika Pergeseran Slider Utama di Halaman Artikel
    function updateSlider(index) {
      currentIdx = index;
      if (sliderWrapper) sliderWrapper.style.transform = `translateX(-${currentIdx * 100}%)`;
      if (sliderBadge) sliderBadge.textContent = `${currentIdx + 1} / ${imagesList.length}`;
    }

    if (sliderPrev && sliderNext) {
      sliderPrev.addEventListener('click', function(e) {
        e.stopPropagation(); // Mencegah pemicu event klik gambar pembuka lightbox
        let newIdx = currentIdx - 1;
        if (newIdx < 0) newIdx = imagesList.length - 1;
        updateSlider(newIdx);
      });

      sliderNext.addEventListener('click', function(e) {
        e.stopPropagation();
        let newIdx = currentIdx + 1;
        if (newIdx >= imagesList.length) newIdx = 0;
        updateSlider(newIdx);
      });
    }

    // Logika Pengendali Gambar di Layar Teater (Lightbox)
    function updateLightboxImage(index) {
      currentIdx = index;
      lightboxImg.src = imagesList[currentIdx];
    }

    function openLightbox(index) {
      updateLightboxImage(index);
      lightbox.style.display = 'flex';
      setTimeout(() => { lightbox.style.opacity = '1'; }, 10);
      document.body.style.overflow = 'hidden';
    }

    const closeLightbox = function() {
      lightbox.style.opacity = '0';
      setTimeout(() => { 
        lightbox.style.display = 'none'; 
        document.body.style.overflow = ''; 
        updateSlider(currentIdx); // Menyinkronkan halaman luar saat teater ditutup
      }, 200);
    };

    // Daftarkan aksi klik perbesaran teater ke setiap dari 4 gambar slide
    slideImages.forEach((img, index) => {
      img.addEventListener('click', function() { openLightbox(index); });
    });

    // Kontrol tombol navigasi internal di dalam Teater Lightbox
    prevBtn.addEventListener('click', function(e) {
      e.stopPropagation();
      let newIdx = currentIdx - 1;
      if (newIdx < 0) newIdx = imagesList.length - 1;
      updateLightboxImage(newIdx);
    });

    nextBtn.addEventListener('click', function(e) {
      e.stopPropagation();
      let newIdx = currentIdx + 1;
      if (newIdx >= imagesList.length) newIdx = 0;
      updateLightboxImage(newIdx);
    });

    closeBtn.addEventListener('click', closeLightbox);
    lightbox.addEventListener('click', function(e) {
      if (e.target === lightbox || e.target === lightboxImg) { closeLightbox(); }
    });

    // Integrasi Pintasan Akses Keyboard (Panah Kiri, Kanan, & ESC)
    document.addEventListener('keydown', function(e) {
      if (lightbox.style.display === 'flex') {
        if (e.key === "Escape") closeLightbox();
        if (e.key === "ArrowLeft") prevBtn.click();
        if (e.key === "ArrowRight") nextBtn.click();
      } else {
        if (e.key === "ArrowLeft" && sliderPrev) sliderPrev.click();
        if (e.key === "ArrowRight" && sliderNext) sliderNext.click();
      }
    });
  });
